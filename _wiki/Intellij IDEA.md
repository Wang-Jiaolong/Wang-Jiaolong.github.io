# 快捷键及快速输入

## 1、Java

**快捷键**

| 按键             | 描述                     |
| ---------------- | ------------------------ |
| `Shift`+`F6`     | 快速重命名相同变量名     |
| `Ctrl`+`Alt`+`L` | 代码格式化               |
| `Ctrl`+`D`       | 复制当前行到下一行       |
| `Ctrl`+`X`       | 删除当前行               |
| `F2`             | 定位到下一个报错的代码段 |
| `Alt`+`Enter`    | 导入包                   |



**快速输入**

| 输入   | 结果                                          |
| ------ | --------------------------------------------- |
| `psvm` | `public static void main(String[] args) {  }` |
| `sout` | `System.out.println();`                       |
| `.var` | 快速补全定义变量的类型                        |
